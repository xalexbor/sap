SAP GUI Connect String - необходима клиенту для подключения к инстанциям. Состоит из след. параметров:

<host> - адрес
<instance number> - номер инстанции
/M/<MessageServerHost> - Адрес Сервера Сообщений
/S/<MessageServerPort> - Порт Сервера Сообщений
/G/<LogonGroup> - Используется для группировки
/WAN - для медленных соединений

Полный пример, как должна выглядеть строка соединения:
sapgui /m/twdf10000/s/3600/g/space

Важные админские транзакции:
SM04, AL08 - Список пользователей
SM51 - Обзор серверов приложений
SM37 - Обзор фоновых заданий
SM50,SM66 - Управление рабочими процессами
SM12 - Управление блокировками
SM13 - Управление обновлениями
SM21 - Анализ системных логов
SM02 - Отправка системных сообщений
RZ20 - Общесистемный мониторинг
RZ03,SMLG,SMMS  - служба сообщений



Задаваемые параметры:
rdisp/rfc_use_quotas - проверка ресурсов
rdisp/tm_max_no - количество одновременного подключенных пользователей
rdisp/rfc_max_login - процент возможных подключений
