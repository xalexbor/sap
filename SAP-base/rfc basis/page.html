RFC - remote function calling - удаленный вызов процедур. Сетевой уровень обмена данными между SAP и non SAP системами. Имеет свой внутренний уровень обработки данных. (Скорее всего xml какой нибудь)

Существует несколько видо протокола:
Synchronous RFC (sRFC) - прямое общение между различными системами и между SAP NETWeaver AS и SAP GUI
Asynchronous RFC (aRFC) - для приямого общения между различными системами параллельной обработки заданий
Transactional RFC (tRFC) - для подлинного асинхронного общения.
queue(d) RFC (qRFC) - расширение tRFC.
Background RFC (bgRFC) - приемник для qRFC&tRFC.
