Расположение данных SAP системы:

/usr/sap/<SAPSID>/[SYS;DVEBMGS;D;SCS]/[global;profile;exe;work;j2ee]

<SAPSID> - каталог, содержащий все данные которые могу быть доступны в SAP.

SYS - содержит информацию,которая доступна со всех инстанций комплекса
DVEBMGS<No.> - каталог ABAP-stack центральной инстанции
D<No.> - каталог диалоговых инстанций
SCS<No.> - Java-stack каталог центральной службы.

global - содержит глобальные данные
profile - содержит профили: по умолчанию, начальный и инстанцный
exe - SAP system kernel.
work - логи и примеры конфигов.
j2ee - содержит информацию про Java узлы.


sapstartsrv - Программа которая читает стартовый профайл в начале начал. Стартовый профайл содержит инструкции по запуску служб инстанций.

sapcpe(Central Patch Environment) - служит для копирования/зеркалирования из глобальной директории в локальную.
